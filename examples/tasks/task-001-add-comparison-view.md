# Task Example 001: Add Comparison View

## Request

Add a comparison view that lets an authenticated team member compare two completed audit runs side by side.

## Requirements

- user can select baseline and target run
- show delta for major metrics
- show changed issues grouped by severity and category
- preserve filter state in the URL
- provide export of comparison summary
- comparison page must be keyboard-usable and understandable without colour alone

## Constraints

- do not redesign the entire dashboard
- do not add a chart library without explicit approval
- do not break existing run detail page
- update docs if user-visible behaviour changes
