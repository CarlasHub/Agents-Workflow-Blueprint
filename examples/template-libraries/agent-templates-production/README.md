# Agent Templates Production Library

This folder contains the extracted contents of the former `agent_templates_production.zip` artifact.

## Purpose

Keep the generated template bundle as real, reviewable Markdown files inside the repository instead of a binary archive.

## Structure

- `General_Core/`
- `Software_Engineering/`
- `Legal_Compliance/`
- `Data_Finance/`
- `Strategic_Ops/`

Each domain contains:

- `prompts/`
- `skills/`
- `contracts/`

## Expected file distribution

- 40 prompt templates
- 30 skill templates
- 30 contract templates
- 100 total Markdown files

## Notes

These files are generated template content and are not tied to runtime code in `src/`.
